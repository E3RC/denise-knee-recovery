# Recovery Agents

These role notes define how ChatGPT and Codex should divide the recovery workflow.

## Roles

- `project-manager.md` keeps the plan, priorities, and schedule consistent.
- `backup-nurse.md` checks meds, reminders, discharge rules, and alert coverage.

## Rules

- Keep PII and secrets out of public pages.
- Do not calculate medication doses.
- Do not replace surgeon or pharmacy instructions.
- Keep reminder logic aligned with the actual discharge packet and logged times.
